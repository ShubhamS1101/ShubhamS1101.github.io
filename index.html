PaisaPhile — Conversational Multi-Agent Market Intelligence

A fully conversational financial advisor agent you can actually argue with. Instead of one-shot answers, it breaks market analysis into four specialized agents — Indicator, Pattern, Trend, and Decision — each producing its own reasoning.

The decision agent synthesizes their outputs, and a dialogue agent explains the logic, answers follow-ups, and defends the recommendation if you challenge it.

All analyses are cached as active market windows, dynamically updating only what's stale. The pattern and trend agents generate candlestick charts, detect classical patterns, draw support/resistance levels, and provide structured insights.

It's not an auto-trader — it's an analytical co-pilot.

HuffZip — Huffman-Based File Compression System

A complete implementation of the Huffman Coding algorithm designed for efficient, lossless data compression. The project combines a high-performance C++ command-line tool for real file compression with an interactive browser-based simulator that visualizes how Huffman trees are constructed and encoded.

The C++ utility enables users to compress and decompress text and files using variable-length prefix codes, handling binary data efficiently at the bit level. It builds optimal frequency-based trees, generates compact encoded outputs, and reconstructs original files through precise decoding.

Alongside the system-level implementation, the web interface — built with HTML, CSS, and JavaScript — provides an intuitive visualization of frequency analysis, tree construction, and code generation. This makes HuffZip both a practical compression tool and an educational demonstration of how entropy-based encoding works in real systems.

ZoionNET — Low-Power Animal Intrusion Detection

An edge-AI system built to reduce human-wildlife conflict in forest-border villages. It runs a compact, low-power neural pipeline directly on embedded hardware to detect and classify animal intrusions in real time.

The system listens continuously using acoustic sensors, but it doesn't waste power. A lightweight detection model stays active in a dormant state, monitoring for meaningful signals. When a potential intrusion is detected, a second classification model activates to identify the species — distinguishing between elephant, cheetah, monkey, or background noise — with over 94% accuracy. This dual-stage architecture ensures both efficiency and reliability.

Under the hood, ZoionNet runs on an Arm Ethos-U55 MicroNPU with quantized YOLOv8-based models optimized for edge inference. Audio is sampled at 8 kHz, converted into mel-spectrograms using STFT, and processed through sliding windows with smoothing techniques to reduce false positives. Once an intrusion is confirmed, alerts are transmitted instantly via cellular connectivity to authorities.

The project received an Honorary Mention from IEEE ComSoc for its technical innovation and social relevance. It is supported by the Kenneth Anderson Nature Society and developed under the guidance of Dr. Teena Sharma at IIT Guwahati.

Rain Rain Go Away — Intelligent Wiper Control System

An analog-digital rain-sensing wiper control system that automatically adjusts wiper speed based on rainfall intensity. Instead of fixed-speed manual control, the system detects how heavy the rain is and switches between three discrete speed levels, making it more responsive and practical for real driving conditions.

A rain sensor continuously monitors precipitation, and its signal is amplified and processed using comparator ICs and digital logic circuitry. Based on the detected intensity, relay switches and an L293D motor driver regulate the wiper motor speed. A 555 timer ensures precise 180-degree wiper rotation, while a semi-circular LED display indicates the active speed level. The system also includes a manual override for emergency control.

Building this project required extensive debugging and iteration. In electronics, nothing behaves ideally — signal noise, voltage variations, and real-world component behavior often demand repeated testing and fine-tuning. Getting stable performance from both analog and digital sections working together took significant effort.

The project secured 3rd place at Kriti 2025, IIT Guwahati's inter-hostel technical competition.
